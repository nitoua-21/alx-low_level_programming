### Project 0x02. C - Functions, nested loops

##Project General Requirements

- Allowed editors:`vi`, `vim`, `emacs`
- All files will be compiled on Ubuntu 20.04 LTS using gcc, using the options `-Wall -Werror -Wextra -pedantic -std=gnu89`
- All files should end with a new line
- Use of the Betty style. It will be checked using `betty-style.pl` and `betty-doc.pl`
- Global variables not allowed
- No more than 5 functions per file
- Standard library not allowed. Any use of functions like `printf`, `puts`, etc… is forbidden
- Use of[_putchar](https://github.com/holbertonschool/_putchar.c/blob/master/_putchar.c) allowed
- The prototypes of all  functions and the prototype of the function `_putchar` should be included in your header file called main.h

Bellow the lists of programs for the tasks:

- `0-putchar.c`:  program that prints `_putchar`, followed by a new line.
