##Project General Requirements

- Allowed editors:`vi`, `vim`, `emacs`
- All files will be compiled on Ubuntu 20.04 LTS using gcc, using the options `-Wall -Werror -Wextra -pedantic -std=gnu89`
- All files should end with a new line
- Use of the Betty style. It will be checked using `betty-style.pl` and `betty-doc.pl`
- Global variables not allowed
- No more than 5 functions per file
- Standard library not allowed. Any use of functions like `printf`, `puts`, etc… is forbidden
- Use of[_putchar](https://github.com/holbertonschool/_putchar.c/blob/master/_putchar.c) allowed
- The prototypes of all  functions and the prototype of the function `_putchar` should be included in your header file called main.h

Bellow the lists of programs for the tasks:

`0-reset_to_98.c`: function that takes a pointer to an int as parameter and updates the value it points to to `98`.
`1-swap.c`: function that swaps the values of two integers.
`2-strlen.c`:  function that returns the length of a string.
`3-puts.c`: function that prints a string, followed by a new line, to `stdout.`
`4-print_rev.c`: function that prints a string, in reverse, followed by a new line.
`5-rev_string.c`: function that reverses a string.
`6-puts2.c`: unction that prints every other character of a string, starting with the first character, followed by a new line.
`7-puts_half.c`:function that prints half of a string, followed by a new line. 
`8-print_array.c`: function that prints n elements of an array of integers, followed by a new line.
`9-strcpy.c`: function that copies the string pointed to by `src`, including the terminating null byte `(\0)`, to the buffer pointed to by `dest`.
